# Heading Accessibility Standards

## Guidelines
- Use semantic `h1–h6` hierarchy (no skipped levels)
- One `h1` per page; subsequent headings nested logically
- Headings describe content that follows

## Test Checklist
- Exactly one `h1` per page
- Heading levels do not skip (e.g., `h2` under `h1`)
- Headings accurately summarize subsequent content
- Visual order matches semantic order
