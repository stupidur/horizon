# Slider Accessibility Standards

## Guidelines
- Use `role="slider"` with `aria-valuenow`, `aria-valuemin`, `aria-valuemax`
- Keyboard support: Arrow/Page/Home/End
- Visible value and focus indicator

## Test Checklist
- Slider has `role="slider"` and correct `aria-*` values
- Keyboard interactions adjust value as expected
- Current value is visible and focus indicator is present
